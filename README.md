# Tease

## Development Setup

### 1. Install global dependencies:

> #### [Node.js](https://nodejs.org/en/download/)

> #### [Harp](http://harpjs.com)

    npm install -g harp

> #### [Gulp](https://gulpjs.com/)

    npm install -g gulp-cli

### 2. Install dev dependencies:

    cd "C:\Users\..."
    npm install

### 3. Run server:

    harp server

### 4. Compile & minify:

    harp compile
    gulp
