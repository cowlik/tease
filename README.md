# Tease

## Development Setup

### 1. Install global dependencies:

> #### [Node.js](https://nodejs.org/en/download/)

> #### Harp

    npm install -g harp

> #### Gulp:

    npm install -g gulp-cli

### 2. Install project dependencies:

    cd "C:\Users\..."
    npm install

### 3. Run server:

    harp server

### 4. Compile & minify:

    harp compile
    gulp
